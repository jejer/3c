Introduction

Symbol Technologies, a world leader in mission-critical mobility solutions, continues its tradition of innovation with the latest software and support utilities for the Symbol Spectrum24 LA-4121 PC Card, LA-4123 PCI Adapter and LA-4137 Wireless Networker. This breakthrough WLAN technology delivers significantly enhanced functionality and mobility management along with lower cost of deployment and ownership. The Spectrum24 11Mbps Wireless LAN Software features a complete, robust security suite that protects mobile data, applications and your network through proven mobile security technologies. Go to: http://www.symbol.com/services/downloads, and click on the Wireless Products link to access Spectrum24 802.11b 11Mbps software and documentation downloads.


Spectrum24 11Mbps Wireless LAN Driver Only Maintenance Release 3.9.71.178 (October 2004)

The new features and resolved issues addressed between the 3.9.68.175 and 3.9.71.178 versions of the Spectrum24 11Mbps Wireless LAN Driver include:

    New Feature

  • WHQL Certification - The Symbol LA-4121 PC Card has been Windows Hardware Quality Labs (WHQL) certified for interoperability with Microsoft Windows XP.

    Resolved Issue

  • A PSK-TKIP Symbol Client can now associate properly to a Symbol ESSID configured for PSK-TKIP and WPA TKIP.



Spectrum24 11Mbps Wireless LAN Driver Only Major Release 3.9.68.175 (February 2004)

The new features and resolved issues addressed between the 3.7.27.51 and 3.9.68.175 versions of the Spectrum24 11Mbps Wireless LAN Driver include:

    New Features

  • WiFi Protected Access (WPA) - Support for WPA security modes, including Pre-Shared Key (PSK) and 802.1x authentication.

  • KeyGuard-MCM - Support for KeyGuard-MCM dynamic encryption key security.

  • Mobile Companion or Driver Only Installation Options - Symbol recommends using Mobile Companion as the adapter management utility. If, however, a third party software supplicant is being used, a driver only installer is available for use with the Windows provided Zero Config utility as the software management tool.

  • Windows Mobile 2003 Operating System Support - Spectrum24 802.11b 11Mbps software now provides driver support for the Windows Mobile 2003 operating system.

    Resolved Issues

  • The Symbol LA-4137 Wireless Networker no longer experiences hangs when using WEP.

  • Symbol LA-4123 PCI adapters are now able to "wake up" properly when set to DHCP.


Known Issues

The following issues remain outstanding or are outstanding with a workaround for both the 3.9.68.175 and 3.9.71.178 releases:

  • If using a Symbol LA-4121 PC card or LA-4137 CF adapter in a Windows XP system, the adapter does not appear as a device that can be safely unplugged or ejected. This does not impact performance or functionality. The adapter can be removed safely at any time.

  • On Windows Mobile 2003 systems using Zero Config, the following error message appears when configuring 802.1x/PEAP authentication: "Cannot log onto the wireless network - requires personal certificate. Contact Network Administrator.". This warning displays even though the authentication is successful. This is a known issue with the Mobile 2003 operating system.

  • A "Call to WZCEnumInterfaces failed to get adapter info" error could display on Windows Mobile 2003 devices after a cold-boot if using PEAP authentication. Cold-boot persistence is not supported for PEAP or any authentication type requiring Certificates.

  • Windows Mobile 2003 device fields clear after about 20 seconds when a PEAP username, password and domain credentials are entered within the Network Logon screen. This is a Windows Mobile 2003 design issue. Every effort should be made to enter the required information within 20 seconds.

  • Do not unzip both the Mobile Companion and Driver Only installer folders to the same subdirectory. The installation will not work properly and one of the folders will need to be removed in order for the desired installer to function correctly.

    Known Issues with Workarounds

  • When using 802.1x authentication with a Funk Odyssey client on a Cisco AP 1200, the client does not authenticate and the AP log displays "Removing MU because of max retries." Increase the max retries limit on the AP. The "Removing MU because of max retries" message could still appear in the log, but the client will be able to authenticate.

  • On HP NX 9010 notebook computers installed with Mobile Companion or driver-only installations using a Symbol LA-4121 PC card adapter, the computer hangs on boot up if the adapter is inserted. The hang is caused by a memory resource conflict. To fix this conflict, install Mobile Companion or complete the driver only installation and remove the adapter. Boot up the computer. Insert the card. Open the Device Manager. Show hidden devices. Select the Symbol LA-4121 PC Card and open its properties. Click on the Resources tab. Select Memory Range Resource. Unselect "Use Automatic Settings" and click "Change Settings...". Change the value fields to a different range until there are no conflicts with other devices. Click OK to save the changes. A dialog box displays prompting "Do you want to Continue?" Click Yes.

  • On HP H2215 iPAQs (using Zero Config), the adapter could stop working. Remove and reinsert the adapter to restore functionality.

  • Computer hangs have been encountered during LA-4121 PC Card installations on HP EVO 800 and 500 notebook computers running Windows 2000 (both Mobile Companion and driver only installations). Before installing Mobile Companion (or just the driver), disable the PCI to USB enhanced host controller driver. Install Mobile Companion (or just the driver). Re-enable the PCI to USB enhanced host controller driver. Open the Device Manager. Select the Symbol LA-4121 PC Card and open its properties. Click on the Resources tab. Unselect "Use Automatic Settings" and click "Change Settings...". Change the value fields to a different range until there are no conflicts with other devices. Click OK to save the changes. A dialog box displays prompting "Do you want to Continue?" Click Yes.

  • On computers using an Intel Centrino, an error message displays during Mobile Companion or driver-only installations. Uninstall the Intel Sebring API before installing Mobile Companion or the driver. This disables the Intel Centrino feature on the computer. Install the Symbol software again.

  • Computer hangs have been identified on HP Omni notebook computers using the LA-4121 PC Card (both Windows XP and 2000 with either Mobile Companion or driver only). To rectify, disable or uninstall the modem before installing Mobile Companion. Install Mobile Companion or just the driver. Re-enable the modem. Open the Device Manager. Select the Symbol LA-4121 PC Card and open its properties. Click on the Resources tab. Unselect "Use Automatic Settings" and click "Change Settings...". Change the value fields to a different range until there are no conflicts with other devices. Click OK to save the changes. A dialog box displays prompting "Do you want to Continue?" Click Yes.

  • An EVMON error could appear when inserting a Symbol adapter. This has been observed when Mobile Companion or the driver was initially installed with a LA-4121 PC Card adapter and then a LA-4137 CF card adapter was used. Close the EVMON error dialog box. This event does not affect performance or functionality.
